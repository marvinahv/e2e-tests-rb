## Process to define/test a feature:

1. Create `.feature` files inside the `/features` folder.

2. Run in the terminal:
```shell
spinach --generate
```

3. Make changes.
4. Run in the terminal:
```shell
spinach
```

See more at [Spinach Github site](https://github.com/codegram/spinach).
